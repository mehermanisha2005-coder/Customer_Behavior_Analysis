# 📊 Customer Shopping Behavior Analysis

## 🔹 Overview
This project demonstrates an end-to-end **data analytics workflow** to analyze customer shopping behavior. It includes data preparation, exploratory data analysis (EDA), data cleaning, SQL querying, dashboard creation, and reporting.

**Objective:**  
To extract meaningful insights from customer data and support data-driven business decision-making.

---

## 📁 Dataset
- Customer shopping behavior dataset  
- Contains both **numerical and categorical features**  
- Includes customer purchase patterns, categories, and trends  

---

## 🛠️ Tools & Technologies

### Python Libraries Used
- pandas → Data loading, cleaning, and analysis  
- sqlalchemy → Database connection and SQL integration  

### Other Tools
- MySQL → SQL analysis  
- Power BI → Interactive dashboard creation  
- Gamma → Presentation (PPT) creation  
- Jupyter Notebook → Development environment  

---

## 🚀 Steps Followed

### 1. Data Preparation in Python
- Loaded dataset using `pandas`  
- Explored data structure using `.info()` and `.describe()`  
- Checked and handled missing values  
- Renamed columns for better readability  

---

### 2. Exploratory Data Analysis (EDA)
- Performed summary statistics analysis  
- Identified patterns and trends using pandas  
- Analyzed data distributions and relationships  

---

### 3. Data Cleaning
- Handled null (missing) values  
- Removed duplicate records  
- Standardized formats (e.g., dates, categories)  

---

### 4. SQL Analysis (MySQL)
- Imported cleaned data into MySQL  
- Wrote SQL queries to:
  - Analyze customer behavior  
  - Perform aggregations  
  - Extract key business insights  

---

### 5. Dashboard Creation (Power BI)
- Built an interactive dashboard  
- Included:
  - KPI cards  
  - Bar charts and line charts  
  - Scatter plots  
  - Filters and slicers for user interaction  

---

### 6. Reporting & Presentation
- Created a detailed report summarizing findings  
- Designed a presentation using Gamma  
- Presented insights in a clear and visual format  

---

## 📊 Results / Insights
- Identified customer purchasing patterns  
- Discovered key trends and relationships in data  
- Improved data quality through cleaning  
- Generated insights using Python and SQL  
- Built an interactive dashboard for easy understanding  

**Impact:**
- Helps businesses understand customer behavior  
- Supports better decision-making  
- Provides clear and visual insights
